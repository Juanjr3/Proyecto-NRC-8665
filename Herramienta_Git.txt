Herramienta git

Esta herramienta es muy útil para guardar códigos, modificar o crear archivos a través de una consola entre otras cosas,
mediante comandos, también se puede subir archivos a una nube que en este caso seria "git hub", en la cual se encontraría
un respaldo en caso de algún inconveniente como daño de disco duro o problemas de Windows u otros problemas, esto se podría 
lograr si antes se vincula remotamente de una carpeta a un repositorio en "git hub", en el cual se puede descargar o clonar
un respaldo de archivos a una carpeta mediante “git bash” la cual nos muestra una consola en la cual podemos clonar o subir 
los archivos con un link del repositorio de "git hub", es muy útil en caso de recuperar versiones anteriores de códigos y archivos.
